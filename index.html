TRAPPIST-1 VR

> OVERVIEW

An interactive VR simulation of the TRAPPIST-1 exoplanetary system, designed to visualize tightly packed planetary orbits, compare relative planet sizes, and explore potential habitability.

> FEATURES

  • Explore all seven TRAPPIST-1 planets in VR with adjustable simulation speed.
  • View scientifically accurate relative planet sizes and orbital periods.
  • Side-by-side size comparison between Earth and each TRAPPIST-1 planet.
  • Per-planet information panels detailing planet type, orbital period, and habitability notes.
  • Free navigation using VR locomotion and interaction controls.

> TECHNICAL NOTES

  • Built in Unity using C#.
  • Implemented custom orbital motion using trigonometric functions (Mathf.Sin, Mathf.Cos) for smooth real-time planet movement.
  • Uses a global time multiplier to support fast-forward and slow-motion simulation.
  • VR interactions and locomotion managed via Unity XR Interaction Toolkit and XR Interaction Manager.
  • Planetary data sourced and cross-referenced from NASA’s Eyes on Exoplanets TRAPPIST-1 simulation.
